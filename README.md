# RezPlayer-1.0
This is AudioPlayer written on PHP

Using WinPHP (DS)

Version 1.0

Folder "ext" must be in same folder, that all libraries/executables

# Basic functionality
1. Play music :p

2. Rewind

3. Play/pause

4. Opening folder with music

5. Auto-play (when one track ends, plays next)

6. Very buggy visualizer

7. Volume
